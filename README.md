# AppSecJobs

This repository provides links to Google Forms and Spreadsheets that allow:
- Application Security job seekers post their desired job and skillset
- Employers to post their Application Security jobs

## Job Seekers' Database

Information is automatically deleted after 2 weeks. 

**Use at your own risk!**: Beware of Job Scams! See the following Resources:

- https://consumer.ftc.gov/articles/job-scams
- https://www.colorado.edu/studentemployment/resources/recognize-avoid-job-scams
- https://business.uccs.edu/resources/career-development-center/resources/job-scams
- https://www.stjohns.edu/life-st-johns/career-development/fraudulent-job-offers-and-email-scams
